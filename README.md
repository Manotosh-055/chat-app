Tech Stack

Client : React JS ||

Server : Node JS, Express JS ||

Database : Mongo DB

Run Locally

Clone the project : 

git clone https://github.com/Manotosh-055/chat-app.git

Go to the project directory : 

cd chat_app

Install dependencies : 

npm install

cd frontend/npm install

Start the server : 

npm run start

Start the Client ->

//open now terminal

cd frontend

npm start
